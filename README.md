## 🚀 Quick Start

### Prerequisites
- **Node.js** 18+ 
- **npm** or **pnpm**

### Installation & Development

```bash
# Install dependencies
npm install

# Start development server (http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Project Structure

```
src/
├── components/        # Reusable React components
├── pages/            # Route pages (Home, Blog, Contact)
├── utils/            # Analytics, tracking, structured data
├── constants/        # Animation configs, constants
├── styles/           # Global styles, Tailwind config
├── assets/           # Images, icons, media
└── data/             # Blog posts, content
```

## 🚢 Deployment

### **AWS Amplify + Route 53**
- Auto-deploys from GitHub on every push
- HTTPS with auto-managed SSL certificates
- Custom domain: **epmartins.com.br**
- CloudFront CDN for global performance
- Zero-downtime deployments

### **Deployment Workflow**
```
Git Push (GitHub) → Webhook → Amplify Build → Deploy to CloudFront → Live
```
